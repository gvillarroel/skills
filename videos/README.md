# Private Video Production

Use this directory for scripts and notes for videos that are not expected to be published as skill resources or GitHub Pages examples.

- Keep one subdirectory per video or tightly managed video series.
- Keep reusable public examples inside the owning skill's `assets/examples/` directory.
- Keep rendered MP4/WebM files, raw frames, contact sheets, and review output under `output/` or ignored local subdirectories.
- Prefer shared scripts inside the relevant video folder when several videos use the same renderer.
