Venn Diagram Examples

Nine replayable D3 examples for using circles as Venn-style visual components. The first row favors labeled storytelling, the second uses explicit rotational symmetry, and the third uses asymmetric cluster layouts such as chains, 3+2 groups, and two blocks of 3 joined by a bridge circle. The 5- and 7-circle versions are overlap diagrams for storytelling, not mathematically complete Venn diagrams with every possible region.
