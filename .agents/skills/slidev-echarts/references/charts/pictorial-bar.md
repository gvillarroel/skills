# Pictorial Bar Charts In Slidev

- **Data shape:** Use ordinary categorical bar values. The pictorial encoding changes the mark, not the data contract.
- **Animation pattern:** Update values and let symbol repetition or symbol clipping change visibly across clicks.
- **Display guidance:** Use repeated symbols only when they make magnitude more memorable. Keep symbols simple and aligned to the axis.
- **Modules:** Register `PictorialBarChart`, `GridComponent`, and `TooltipComponent`.
- **Pitfalls:** Decorative symbols can hurt precision; choose bar charts when exact comparison matters.
