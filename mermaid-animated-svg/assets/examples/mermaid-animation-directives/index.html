Compact Language

Diagram directive notes

1. Select

Bind rendered Mermaid elements with id:, #id, text:"...", .class, role:, or narrow css:"...".

2. Place

Name SVG points from anchors, paths, xy(), mid(), and offset(). Use marks for motion, not Mermaid layout rewrites.

3. Orchestrate

Use at, named actions, .end, and shared timestamps to synchronize several diagrams or parallel branches.

4. Extend Later

Keep semantic selectors like state:, edge:A->B, plot(), and message: documented as future syntax.
