# Effect Scatter Charts In Slidev

- **Data shape:** Use scatter tuples with stable names and quantitative x/y coordinates.
- **Animation pattern:** Enable ripple effects after the first click or only on emphasis. Update x/y positions like a normal scatter plot.
- **Display guidance:** Use effect scatter sparingly for priority points, alerts, or live-system emphasis.
- **Modules:** Register `EffectScatterChart`, `GridComponent`, and `TooltipComponent`.
- **Pitfalls:** Ripples can obscure dense plots; reduce symbol size or switch to plain scatter for analytical reading.
