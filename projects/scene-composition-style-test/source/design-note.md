# Scene Composition Style Test

## Concept Claim

The video tests the new scene-composition direction style: every shot has a clear visual job, a named composition family, visible hierarchy, depth layers, and a semantic motion phase.

## Mechanic

A cache rollout moves from uncertainty, through branch mechanics, into measurable proof, then into a final action. The viewer should understand the system by following one request token, not by reading a paragraph.

## D3 Composition Revision

The first render used a repeated diagnostic sweep behind each scene to keep the MP4 active during held moments. That solved a technical validation problem but weakened the composition language because every scene shared the same background motion. This revision removes that repeated sweep and makes motion come from each scene's own D3 diagram structure.

## Candidate Metaphors

- **Pipeline**: good for sequence and handoff, but too linear for the branch comparison.
- **Split mechanism**: good for the cache hit/fallback scene, but too narrow for proof and CTA.
- **Composition sampler**: each scene uses a different composition family while preserving one request-token vocabulary.
- **D3 armature sampler**: each scene uses a different D3-friendly structure so motion comes from the diagram itself.

Chosen metaphor: D3 armature sampler. It tests a thirds force-network, a flow-spine branch, a golden/root before-after chart, and a radial rosette checklist in one short piece.

Rejected reuse: a repeated background sweep. It kept pixels changing but did not preserve a semantic role across scenes, so it read as decoration.

## Visual Vocabulary

- Ink text and hairline grid: technical structure.
- Teal request token: the tracked object.
- Blue/green path: cache hit and proof.
- Coral path: origin fallback or uncertainty.
- Yellow accents: active decision or proof.
- Panels and rails: fixed composition structure, not web chrome.
- D3 links, bars, arcs, and scales: scene-specific mechanics, not decorative background motion.

## Narration Split

This style test is silent. Minimal words are used as data labels and action labels only.
